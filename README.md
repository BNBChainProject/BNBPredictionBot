# BNBPredictionBot
Prediction Bot Supported Platform

- PancakeSwap (BNB) 
- CandleGenie.io ( BNB , BTC , ETH ) 
- Dogebets.gg (BNB) 
- PRDT Finance (BNB , ETH)

Requirement
- Ubuntu 20.04

How To Use 
- Unzip Release
- chmod +x main
- ./main
